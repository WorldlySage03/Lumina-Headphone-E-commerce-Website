# Lumina Audio 🎧

This is a frontend mockup for a fictional headphone brand called Lumina Audio. It's built with plain HTML and CSS to practice responsive design, glassmorphism, and CSS Grid (specifically the Bento grid on the Features page).

## Features

- **Modern Design**: Built using radial gradients, glassmorphism, and some basic micro-interactions.
- **Bento Grid**: The 'Learn More' page uses a responsive CSS grid layout.
- **Responsive**: Works on both desktop and mobile devices.
- **Zero Dependencies**: Just vanilla HTML and CSS, so there are no build tools or frameworks needed.

## Running Locally

Because these are just static files, there's no build step required.

1. Clone or download the repository.
2. Open `index.html` in your browser.
